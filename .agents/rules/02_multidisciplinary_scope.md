# Rule: Multidisciplinary Scope

## Overview
Tasks in this repository span multiple engineering disciplines.
Agents must understand which domain they are working in and
apply the appropriate standards for that domain.

## Domains

### Data Engineering
- Data modeling (Star Schema, Data Vault)
- ETL/ELT pipelines
- Streaming architectures
- Data warehousing

### Machine Learning Engineering
- MLOps and model lifecycle management
- Model training pipelines
- Feature stores
- Model deployment and serving architectures

### AI Engineering
- Generative AI integration
- RAG (Retrieval-Augmented Generation) architectures
- LLM orchestration and prompt engineering
- Vector databases and embedding stores
